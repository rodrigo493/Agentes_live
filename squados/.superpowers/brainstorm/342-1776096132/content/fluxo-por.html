Como funciona o fluxo de produção?

Quando o admin adiciona processos do catálogo, eles são adicionados para quem?

A

Por usuário individual

Cada operador tem seu próprio conjunto de processos no fluxo. João pode ter Processo A → B → C e Maria ter D → E. O admin escolhe os processos de cada um separadamente.

Como funciona hoje com assigned_to

B

Por setor

Todos os operadores de um setor veem o mesmo fluxo de processos. Admin define o fluxo do "Setor Montagem" e todos que pertencem a ele veem automaticamente.

Menos trabalho de configuração, menos flexibilidade individual

C

Base por setor + customização individual

O setor tem um fluxo padrão, mas o admin pode adicionar ou remover processos específicos para cada operador além do padrão do setor.

Mais poderoso, mas mais complexo de implementar
