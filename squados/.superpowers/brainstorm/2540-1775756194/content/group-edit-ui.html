Como editar um grupo no Workspace?

Só Admin e Master Admin veem o controle de edição

A

Botão de engrenagem (⚙) ao lado do grupo na sidebar

Abre um modal com abas: Informações / Membros

  • Acesso direto sem entrar no chat
  • Ícone aparece só para admin ao passar o mouse
B

Botão "Editar grupo" no cabeçalho do chat do grupo

Ao abrir um grupo, admin vê um botão de edição no header do chat.

  • Contexto claro — você já está no grupo
  • Precisa entrar no chat para editar
